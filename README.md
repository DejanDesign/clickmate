Customization

ClickMate includes a number of customization options that allow you to modify the size, color, and style of your buttons. To customize your buttons, simply modify the index.css file as needed.

Installation
To install ClickMate via npm, run the following command:

npm install clickmate

Or, if you are using yarn, run the following command:

yarn add clickmate
